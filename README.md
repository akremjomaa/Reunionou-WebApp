# Reunionou-WebApp (Atelier n°2)

*Projet réalisé du 27 mars au 5 avril 2023*

## Récupérer le projet 

```sh
git clone git@github.com:akremjomaa/Reunionou-WebApp.git
cd Reunionou-WebApp
```

## Initialisation

### Installation des dépendances

```sh
npm install
```

### Tester le site

```sh
npm run dev
```
